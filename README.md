# IBONARIUM :: ECONOMY OSINT

A real-time financial OSINT dashboard with a cyberpunk aesthetic.

## Features
- **Real-time Data**: 100% live data integration from various financial APIs.
- **Crypto Ticker**: Live prices for Bitcoin, Ethereum, Solana, and more.
- **Fiat & Metals**: Currency exchange rates and precious metals prices.
- **OSINT News**: Latest news from the crypto and financial world.
- **Anomaly Radar**: Scans for market anomalies and risks.
- **Cyberpunk UI**: Modern, glassmorphism-inspired design with CRT effects.

## Technologies
- HTML5 / CSS3 (Custom Vanilla CSS)
- JavaScript (ES6+)
- TradingView Widgets
- Real-time APIs (CoinGecko, CryptoCompare, ExchangeRate API, Alternative.me, Llama.fi)

## Installation
1. Clone the repository.
2. Open `index.html` in any modern web browser.
